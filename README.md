# Backgammon Game 🎲

A desktop implementation of the classic Backgammon game, developed as a core project for my Object-Oriented Programming (OOP) course at ULBS.

### 💻 Technical Overview
- **Language:** C++
- **Environment:** Embarcadero / C++ Builder
- **Paradigm:** Object-Oriented Programming (Inheritance, Polymorphism, Encapsulation).

### 🕹️ Features
- **Full Game Logic:** Implementation of standard Backgammon rules, including doubling cubes and specialized move validation.
- **Multiplayer:** Local/LAN multiplayer functionality.
- **UI:** Graphical interface built using the VCL framework.

### 🧠 Lessons Learned
This project was a deep dive into managing complex game states and ensuring clean class hierarchies in C++. It focuses on the separation of game logic from the user interface.
